# Memory Games 🧠

Three memory games with a dark 3D-themed UI. Built with Three.js + vanilla JS.

<img width="2560" height="1358" alt="brave_screenshot" src="https://github.com/user-attachments/assets/824ef2dc-d8f3-4870-a0d6-c068df48475d" />

## Games

### 1. 🔢 Remember the Numbers
All digits appear at once. Timer counts down (1min for 10, 2min for 25, 5min for 50, 10min for 100). Hit "I Remember" when ready — then type every digit in order.

<img width="2560" height="1358" alt="brave_screenshot (1)" src="https://github.com/user-attachments/assets/1e914e3c-4d0e-4e73-8a78-42491367cd46" />

### 2. 📝 Remember the Words
All words displayed at once with the same timer system. Type every word back in order — correct answers shown below each input after checking.

<img width="2560" height="1358" alt="brave_screenshot (2)" src="https://github.com/user-attachments/assets/9730104d-aded-4860-9eee-97de01fca8c8" />

### 3. 🎨 Your Challenge
Create your own — planets, capitals, vocabulary, anything. Type items, set memorise time, play.

<img width="1761" height="1328" alt="brave_screenshot (3)" src="https://github.com/user-attachments/assets/6788d55b-4603-4222-9541-a36f2c78fae8" />

## Play

Open `index.html` in any browser. No build step — just serve the folder.

## Deploy

Push to GitHub Pages, Vercel, Netlify — static HTML, zero config.
